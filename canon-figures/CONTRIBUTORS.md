## Coverage notes

- **Total figures (pass 1):** 76
- **Branches covered:** 10 (7 strict canon + 3 expansion: tradition, art, earth)
- **Earliest:** Homer / Lao Tzu / Aristarchus tradition (c.8th–6th century BCE)
- **Latest:** Chomsky (1928– )
- **Posthumous / delayed-canon entries:** Wegener, Mendel, Milanković, Marie Tharp, Galois, Cantor, Boltzmann (worth flagging — the rejection-then-vindication pattern is itself a historical signal)

### Diversity of traditions
Pass 1 deliberately spreads across Greek, Hellenistic, Roman, Arab, Persian, Indian, Chinese, Japanese, Danish, Italian, French, German, English, Scottish, Irish, Russian, Polish, Belgian, Hungarian, Serbian, Argentine, and American traditions. This is **partial, not exhaustive** — pass 2 should pull in more figures from sub-Saharan Africa, the Islamic Golden Age beyond the four already represented, Mesoamerica, Vedic mathematics beyond Brahmagupta and Aryabhata, and the modern Global South.

### Women
Pass 1 includes Sappho, Émilie du Châtelet, Marie Skłodowska-Curie, Emmy Noether, Rosalind Franklin, Lynn Margulis, Vera Rubin, and Marie Tharp — eight figures historically under-credited relative to their contributions. This is also **partial**; pass 2 should add (at minimum) Hypatia, Ada Lovelace, Sofia Kovalevskaya, Lise Meitner, Chien-Shiung Wu, Henrietta Leavitt, Cecilia Payne-Gaposchkin, Dorothy Hodgkin, Barbara McClintock, Mary Anning, and Caroline Herschel.

## How to add a figure

1. Read `SCHEMA.md` for the card contract.
2. Add the markdown card to the appropriate branch file (`NN-name.md`), in roughly chronological order.
3. Add the JSON entry to `figures.json`.
4. Add the line here in `CONTRIBUTORS.md` (both views).
5. Note the addition in `_pass-log.md` under the current pass.
