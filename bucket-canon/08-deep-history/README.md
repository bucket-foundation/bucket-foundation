# 08-deep-history — Canon Branch

**Scope**: Foundation-tier primary evidence about Earth's Holocene and late-Pleistocene human history and planetary events — dating methods, primary archaeological reports, geological and geomagnetic primary data, archaeoastronomy primaries.

**Not in scope** (goes to `research-landscape/` instead):
- Alt-history synthesis / popular commentary (Hancock, Carlson, UnchartedX — goes to `research-landscape/history-commentariat/`)
- Comparative-mythology pattern-hunting (goes to `research-landscape/myth-patterns/`)
- Hypothesis-tier claims without peer-reviewed dating

**Promotion rule**: landscape → canon requires at least one peer-reviewed primary source with replicable method. Retractions demote back.

## Structure

```
08-deep-history/
  README.md            (this file)
  CANON_INDEX.md       (authoritative manifest — if not in index, not canon)
  sites/               (global ancient-site index, dated primaries)
  cataclysms/          (event catalog, peer-reviewed primaries)
  archaeoastronomy/    (primary papers on alignments, precession records)
```

## Branch editorial stance

This branch is politically contested because popular alt-history and academic archaeology disagree on dates, methods, and interpretations. The canon's only job is to record what peer-reviewed primary evidence actually says — including where primary methods disagree with each other (dating-method disputes are in-canon; the Hancock-vs-academia debate is landscape).
